Oblig 2
=======
OsloMet brukernavn: s374200

GitHub brukernavn: Omer-Gndz

Github repo URL: https://github.com/data1700-v23/oblig-2-Omer-Gndz

Fullt navn: Ömer Gündüz

Kort beskrivelse av applikasjon (5-10 setninger): Denne koderen genererer kinobillettbestillingsskjema.  Koden lar brukeren legge inn informasjonen og til slutt kjøpe billettene med et klikk på en knapp.  Ved bruk av JavaScript viser koden en passende feilmelding hvis brukeren ikke skriver inn eller feilskriver informasjon i noen av feltene.  Den legger også til navnet på filmen som brukeren velger, hvor mange billetter han har kjøpt, og personlig informasjon til en liste, og oppretter deretter en seksjon der brukeren kan se denne informasjonen.  Det er også en knapp for brukeren for å slette alle billetter.
